Axomic Ai Is A Free Local Artificial Intelligence that Are Made by Axom. 
there is currently no trained model capable of doing anything. 

This Ai is designed to use PyTorch without the need for Transformers/TensorFlow.

this ai is fully manageable/trainable by the user (you) to be able to make if great! 

How? 
you can find datasets like Wikipedia dumped, Reddit dumped, etc
search "Training Datasets" to find some huge datasets 🔥


Install Requirements :
`pip install -r requirements.txt`

Run Manager (Training, Chatting with the Ai, export & import, undo & redo, etc) :
`python manager.py`

Run The Ai LocalHosted Web Interface :
`python app.py`
