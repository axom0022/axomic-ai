Axomic Ai Is A Free Local Artificial Intelligence that Are Made by Axom. 
this ai is currently under data training and there is currently no trained model capable of doing anything. 

This Ai is designed to use PyTorch without the need for Transformers/TensorFlow.

if you would like to help in making this ai great, send me training datasets files or URLs for files and I will train the ai, or train the ai yourself and send the exported model. 
the first ever model for this ai will be named "Vuaxon A1" 

Install Requirements :
`pip install -r requirements.txt`

Run Manager (Training, Chatting with the Ai, export & import, undo & redo, etc) :
`python manager.py`

Run The Ai LocalHosted Web Interface :
python app.py
